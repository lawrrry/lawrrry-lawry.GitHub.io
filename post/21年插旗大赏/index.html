21年插旗子大赏

2021-02-20 5 min read # 计划总结

前几天博客搭建终于完成,也发了第一篇文章.其实总是有点不尽人意的,比如不能评论,比如访问有限制
好歹也是落成了呀.第一步踏出去了,在过程中慢慢学习吧(比如这个markdown输入我就不太懂啥意思)
追求完美的话,可能开学前也写不上一篇

那么2020过去了,2021要做什么呢
过去两年难免浑浑噩噩啊,一切的一切给我的感觉可以浓缩成三个字----不确定
人活着总得是要有些盼头的,比如读书(finally)
但不能光盼不做,不可重蹈覆辙(指大学四年碌碌无为),那么就来计划吧

对于"去学习"这件事的准备

  1. SQL:
    做到能明白这是一个什么东西,如果要用是怎么用的,有什么用.
    具体发挥他真正作用的工作就开学后再精进学习.
  2. python:
    之前小小地学了一段时间,大概到了能打包一个小游戏.exe的程度吧(没有尽善尽美哈)
    但还没理解python是如何发挥效能的,使用也不熟练,忘差不多了.
    希望自己能学会抓取数据,简单的清洗数据,生成excel表格和简单的统计处理
  3. excel:
    课程已经看了大部分,有空抽一点时间把剩下的看完再把自己的笔记整理一下,平时工作尽量
    用上去就好.
  4. 看书:春节前就断断续续在看了,目前手上数分科普向和技能向的书比较多,争取在开学前看完(精读)
    1. 赤裸裸的统计学(在读
    2. 金字塔原理
    3. 利用python进行数据分析
    4. 增长黑客
    5. SQL必知必会

生活一些比较杂的

体态:
年前跑步一圈下来,身体冷却之后,尾椎骨会很痛,搜查资料后发现也许和骨盆前倾(现在发觉可能是后倾)有关.
好久没有打球啊游泳啊什么的运动了,再加上久坐打游戏或者办公,我感觉我现在身体的体态非常差
目前搜罗了好些用于矫正康复的视频,归纳了七八个最常出现的动作.睡前起床做一次,可以矫正体态还能拉伸酸痛的地方.
目标是:骨盆恢复中立位,走路姿势,坐姿调整到更健康的样子.

阅读:
除了专业相关的,还想看一些别的书,提升下自己狭隘的思想.
除开小时候看冒险小虎队,自己似乎并没有阅读这个习惯或者说爱好,我要改变!
偶尔在看别人朋友圈啦,听别人说话啦或多或少会感觉到一些人的底蕴会明显高于自己,无一不是三言两语就能让我感受到他们的阅读量的.
既然没能力去外面更大更大的世界看,那就让书代步吧.
今年定个小一点的目标,就10本书吧(关公插旗)

抵抗自卑:
一直都是一个很在乎别人想法的人,这种在乎总是先入为主的影响我待人接物的表现.
比如如果是被我认为厉害的(无论是能力还是颜值)人,在接触期间我会很希望与他们在同一水平线上,可实际不行.这种时候我就做不了自己,行为会非常不舒展.
我觉得这是因为我一开始就带着目的性,带着被认可的目的.进而导致自己的自卑. 其实吧,我也是有可以被认可的点的, 只是和自己的追求有偏差.
我希望在自己想通之后,尽量去克服这种态度,做自己就好,不带过多包袱去社交或工作学习.
